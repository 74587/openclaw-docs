# 架构计划专题

这里放维护者计划和设计草案。它们不是新手教程，但能帮助开发者理解 OpenClaw 为什么这样演进。

- [Codex Context Engine Harness](/tutorials/plan/codex-context-engine-harness)
- [Channel UI Presentation Refactor](/tutorials/plan/ui-channels)

## 怎么读这些计划

计划页不是承诺每个细节都已经上线。它们更像设计草案，解释维护者正在把系统往哪里整理。

普通用户优先读教程；插件作者和贡献者再读计划页，用来理解未来接口可能怎么演进。
