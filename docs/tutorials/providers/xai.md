# xAI

xAI 可用于 Grok 模型、Grok Search、X 搜索和远程 code execution 等能力。

配置：

```bash
export XAI_API_KEY="..."
```

常见用途：

- Grok 模型回复。
- [Grok Search](/tutorials/tools/grok-search)。
- [Code Execution](/tutorials/tools/code-execution)。

如果你只想做网页搜索，先看 Grok Search；如果想跑远程 Python 分析，再看 Code Execution。
