# Skill Workshop

Skill Workshop 帮你把经验沉淀成技能。技能像给 Agent 的小教程，告诉它某类任务应该怎么做。

适合：

- 固定写作格式。
- 团队流程。
- 常见排查步骤。
- 特定 API 使用方法。

继续阅读：[技能 Skills](/tutorials/tools/skills)。

## 技能和插件的区别

技能教 Agent 怎么做事，插件给 OpenClaw 增加新能力。
如果你只是想让 Agent 按固定格式写文档，用技能。
如果你要接一个新 API 或新通道，用插件。
