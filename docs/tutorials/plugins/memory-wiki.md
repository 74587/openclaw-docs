# Wiki 记忆插件

Wiki 记忆插件方向用于把结构化知识、团队文档或 wiki 内容变成 Agent 可检索资料。

关键问题：

- 哪些页面能索引？
- 谁有权限看？
- 多久同步一次？
- 搜索结果如何进入上下文？

不要把所有公司 wiki 一股脑丢给 Agent。先从小范围开始。

## 新手路线

1. 先选一小批非敏感文档。
2. 跑一次索引。
3. 用 `openclaw wiki search` 搜几个问题。
4. 确认结果准确，再扩大范围。

Wiki 记忆适合稳定知识，不适合临时聊天碎片。
