# SDK Entrypoints

入口文件是 Gateway 加载插件 runtime 的起点。

写入口时注意：

- 不要在 import 阶段做重活。
- 注册逻辑保持可预测。
- 配置缺失时给清楚错误。
- 不要在加载时偷偷联网或读敏感文件。

重活应该等真正使用能力时再做。

## 为什么 import 阶段要轻

Gateway 启动时会发现和加载很多插件。如果每个插件一加载就联网、读大文件或初始化重资源，启动会变慢，也更容易因为一个插件拖垮整体。

入口文件应该主要做注册：告诉 OpenClaw“我能提供什么”，真正干活等能力被调用时再执行。
