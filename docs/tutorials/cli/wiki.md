# `openclaw wiki`

`wiki` 是 Wiki 或知识库相关命令，适合把文档资料变成 Agent 可检索知识。

```bash
openclaw wiki status
openclaw wiki doctor
openclaw wiki search "deployment"
```

## 什么时候用

- 想把 Obsidian、Markdown 或团队文档接入记忆。
- Wiki 搜索没有命中，需要检查索引。
- 想让 Agent 能查某批长期知识。

## 新手提醒

Wiki 更像“资料库”，memory 更像“记忆系统”。两者可以配合，但不要把所有临时聊天都塞进 Wiki。

继续阅读：[Wiki 记忆插件](/tutorials/plugins/memory-wiki)。
