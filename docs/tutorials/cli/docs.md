# `openclaw docs`

`docs` 用来从终端打开或搜索 OpenClaw 文档。

```bash
openclaw docs
openclaw docs gateway token secretref
openclaw docs browser existing-session
```

## 什么时候用

- 在终端里忘了命令下一步怎么写。
- 想快速搜索某个配置名。
- 想把官方文档入口打开给浏览器。

如果你是中文读者，也可以直接从本站的 [教程中心](/tutorials/) 开始。

## 新手小技巧

搜索时不要只搜一个词。比如不要只搜 `token`，可以搜：

```bash
openclaw docs gateway token secretref
openclaw docs telegram pairing
openclaw docs plugins install
```

多给几个关键词，结果通常更准。

如果官方文档和本站写法不一样，优先理解本站教程的解释层，再用官方文档确认最新参数。本站目标是帮你看懂，官方文档目标是精确列全。
