# `openclaw nodes`

`nodes` 用来查看和管理连接到 Gateway 的节点。节点可以是桌面 App、手机、远程机器或轻量 node host。

节点可以提供摄像头、屏幕、位置、音频、Canvas 等能力。

```bash
openclaw nodes list
openclaw nodes status
```

## 什么时候用

- 想知道哪些设备已经连上 Gateway。
- 某个工具要在远程机器执行，需要确认节点在线。
- 手机或桌面 companion 没反应。
- 节点能力没有出现在 Agent 可用工具里。

## 新手怎么理解

Gateway 是大脑，节点像手脚和眼睛。节点不一定负责聊天，但它能提供设备能力。
