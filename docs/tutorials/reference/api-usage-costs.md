# API Usage Costs

API 费用主要来自模型、搜索、图片、视频、语音和云服务。

省钱建议：

- 给日常通道用便宜模型。
- 给困难任务手动切强模型。
- 控制视频和图片生成次数。
- 使用提示词缓存和上下文修剪。

继续阅读：[费用和用量](/tutorials/reference/costs-usage)。

## 新手先做三件事

1. 给日常聊天设置一个便宜稳定的默认模型。
2. 视频、图片、TTS 这类高成本能力先手动触发。
3. 定时任务设置低频率，观察几天再放开。

费用异常时，先看任务、日志和 provider 控制台账单。
