# Credits

Credits 记录项目使用、致谢或来源相关信息。

如果你在公司或产品中使用 OpenClaw，建议同时查看许可证、依赖和第三方服务条款。

## 为什么这页重要

OpenClaw 不是孤立存在的。它会连接模型 provider、聊天平台、浏览器工具、媒体服务和开源依赖。
Credits 帮你知道哪些能力来自哪里，也方便团队做合规和致谢。

## 新手提醒

如果你只是个人自用，不需要每天看 credits。
如果你要把 OpenClaw 放进公司流程，至少确认许可证、第三方 API 条款和数据处理要求。
