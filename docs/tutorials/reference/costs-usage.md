# 费用和用量

OpenClaw 本身负责调度，真正费用通常来自模型、搜索、图片、视频、语音和云服务器。

容易花钱的地方：

- 大模型长上下文。
- 视频生成。
- 图片批量生成。
- TTS 长文本。
- 频繁 web search。
- 云端 Gateway 和持久磁盘。

建议先设置小模型、低频率、短任务，再逐步放开。

## 省钱小抄

- 默认模型选便宜稳定的。
- 复杂任务再临时切强模型。
- 媒体生成先人工确认。
- 群聊不要无限保留上下文。
- 自动化任务先低频跑。

如果账单突然变高，先查最近的 cron、tasks、图片/视频生成和 web search 调用。
