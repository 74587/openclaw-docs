# 威胁模型 Atlas：OpenClaw 常见风险地图

OpenClaw 的风险主要来自这些边界：

- Gateway 暴露到哪里。
- 谁能使用通道。
- Agent 能调用哪些工具。
- 密钥怎么保存。
- 节点怎么配对。
- 浏览器和命令执行是否受控。

排查时不要只问“有没有密码”，还要问“拿到这个入口的人能做什么”。
