# GPT-5.5 Codex Agentic Parity：给用户看的理解版

这个主题关注 Codex 风格 Agent 在 OpenClaw 里能否保持接近原生体验：工具调用、长任务、文件修改、会话引导、权限和结果投递是否一致。

普通用户只需要记住：

- 先用默认 runtime。
- 代码任务再考虑 Codex runtime。
- 如果外部 harness 走 ACP，配置和权限会更多。

继续阅读：[Agent 运行时](/tutorials/concepts/agent-runtimes)、[ACP Agents](/tutorials/tools/acp-agents)。
