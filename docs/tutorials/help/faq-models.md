# 模型 FAQ

## OpenClaw 自带模型吗？

不自带大模型。OpenClaw 负责 Gateway、通道、工具和会话，真正回答问题的是你配置的模型提供商。

## 新手选哪个？

能稳定访问 OpenAI 或 Anthropic，就先选其中一个。国内网络更适合 Qwen、Moonshot、GLM、MiniMax 等。

## 可以换模型吗？

可以。你可以在配置或 Control UI 中切换，也可以配置 fallback。

## 本地模型能用吗？

能。最容易从 [Ollama](/tutorials/providers/ollama) 开始。
