# Oracle Cloud：Always Free ARM 路线

Oracle Cloud Always Free ARM 可以低成本运行 OpenClaw，但注册、容量和 ARM 兼容性可能更麻烦。

适合：

- 想尽量 0 月费。
- 能接受 ARM。
- 愿意处理云平台细节。

建议用 Tailscale 访问 Gateway，不要裸露公网控制台。

继续阅读：[Oracle 安装](/tutorials/installation/oracle)。

## 新手提醒

Always Free 的吸引力是便宜，但代价是排障成本更高。常见问题包括容量抢不到、ARM 依赖不兼容、磁盘或内存偏小。

如果你只是第一次学习 OpenClaw，普通 VPS 或本机 Linux 会更省心。等主流程熟了，再尝试 Oracle Cloud。
